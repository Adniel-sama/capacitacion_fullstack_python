# Gestor de Biblioteca -- Instrucciones

## Requisitos

-   Python 3 instalado en el sistema
-   Archivo `biblioteca.txt` en la misma carpeta que el script

## Cómo ejecutar el programa

1.  Coloca el archivo `gestor_biblioteca.py` y `biblioteca.txt` en la
    misma carpeta.
2.  Abre una terminal o consola en esa carpeta.
3.  Ejecuta el script

## Uso general

-   Selecciona una opción del menú ingresando su número.
-   Los cambios realizados (agregar, eliminar, prestar, devolver) se
    guardarán en `biblioteca.txt` al cerrar el programa.
