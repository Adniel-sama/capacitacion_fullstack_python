## **Programa Especial “Talento Digital Para Chile”, Becas Laborales 2024**
    
#### **Curso Desarrollo de Aplicaciones Web Full Stack Python Trainee**
    
**Competencia del Plan Formativo:** Desarrollar aplicaciones web Full Stack utilizando tecnología Python que den solución a las necesidades de la organización, y aplicando las buenas prácticas de la industria para obtener un producto con niveles de calidad requeridos.   

Actividades: 9
    
- ### Módulo 1: Orientación al perfil y metodología del curso (18 Horas) / ELE069945089565
    
    **Descripción del Módulo:** Este módulo tiene como propósito familiarizar al participante con el perfil laboral asociado al área de Tecnologías de la Información (TI), proporcionando una clara comprensión de las competencias técnicas y personales requeridas en el mercado actual. Además, se exploran en detalle los diferentes módulos que componen el plan formativo, enfatizando las competencias a desarrollar, las herramientas necesarias, los productos esperados y su relevancia para construir un portafolio profesional. Asimismo, el módulo presenta la metodología Bootcamp, destacando su importancia para el desarrollo de habilidades clave como el trabajo en equipo, autoaprendizaje, tolerancia a la frustración y ética laboral. Finalmente, se introduce a los participantes en el uso práctico de herramientas de Inteligencia Artificial (IA) enfocadas a mejorar la productividad en programación, abordando conceptos básicos de algoritmos y seudocódigo para facilitar el aprendizaje y desarrollo eficiente en el ámbito TI.
    
    **Competencia del Módulo:** Aplicar las competencias del perfil laboral y la metodología bootcamp en el contexto de la industria TI.
    
    Actividades: 31
    
- ### Módulo 2: Fundamentos del desarrollo Front-End (72 Horas) / ELE069945089566
    
    **Descripción del Módulo:** La implementación de páginas web básicas responsivas es fundamental en el mundo actual, donde los dispositivos de acceso a Internet varían en tamaño y capacidad. Utilizando HTML, CSS y JavaScript, los desarrolladores pueden crear sitios web que se adapten automáticamente a diferentes resoluciones de pantalla y se vean bien en dispositivos móviles y de escritorio. Para lograr esto, es esencial seguir las buenas prácticas de la industria, como el uso de etiquetas semánticas de HTML, la aplicación de metodologías CSS como BEM o SMACSS, y escribir código JavaScript modular y eficiente. Además, el enfoque "mobile-first" y el uso de media queries en CSS garantizan una experiencia de usuario óptima, independientemente del dispositivo utilizado. Al cumplir con estos estándares, los desarrolladores pueden garantizar que sus sitios web sean accesibles, atractivos y funcionales para una amplia variedad de usuarios y contribuir al éxito de su negocio o proyecto en línea.
    
    **Competencia del Módulo:** Desarrollar páginas web básicas responsivas utilizando html, css y javascript de acuerdo a los requerimientos y acorde a las buenas prácticas de la industria.
    
    Actividades: 83
    
- ### Módulo 3: Fundamentos de programación en Python (72 Horas) / ELE069945089567
    
    **Descripción del Módulo:** Este módulo introduce al participante en los fundamentos del lenguaje de programación Python, proporcionando las bases necesarias para la construcción de software de baja complejidad orientado a solucionar problemas comunes en la industria tecnológica. Se exploran conceptos clave como sintaxis básica, estructura e identación, tipos de datos y operaciones fundamentales del lenguaje. Los participantes aprenderán a utilizar estructuras condicionales e iterativas para controlar el flujo de sus programas, aplicando ciclos y sentencias como IF, FOR y WHILE. Además, se aborda el manejo eficiente de estructuras de datos tales como listas, tuplas, conjuntos y diccionarios para modelar problemas del mundo real. Finalmente, se profundiza en el uso de funciones para promover la reutilización del código, destacando aspectos como parámetros, retorno y alcance de variables, esenciales para desarrollar programas organizados y eficientes.
    
    **Competencia del Módulo:** Codificar piezas de software de baja complejidad utilizando lenguaje Python para resolver problemáticas comunes de acuerdo a las necesidades de la industria.
    
    Actividades: 51
    
- ### Módulo 4: Programación avanzada en Python (56 Horas) / ELE069945089568
    **Descripción del Módulo:** Este módulo profundiza en conceptos avanzados de programación en Python, orientándose específicamente al paradigma de orientación a objetos para resolver problemas de mediana complejidad. Se abordan conceptos esenciales como clases, objetos, atributos y métodos, así como los principios fundamentales de abstracción y encapsulamiento. Los participantes aprenderán a implementar programas que utilicen colaboración y composición entre objetos, además de aplicar técnicas como sobrecarga de métodos, herencia y polimorfismo para mejorar la eficiencia y modularidad del código. Asimismo, se estudiarán herramientas para el manejo y control de errores y excepciones, permitiendo un código robusto y confiable. Finalmente, el módulo enseña a gestionar archivos externos mediante lectura y escritura, proporcionando así habilidades esenciales para la manipulación y almacenamiento eficiente de datos desde programas Python.
    
    **Competencia del Módulo:** Codificar piezas de software de mediana complejidad en lenguaje Python utilizando paradigmas de orientación a objetos para resolver problemáticas de acuerdo a las necesidades de la organización.
    
    Actividades: 41
    
- ### Módulo 5: Fundamentos de bases de datos relacionales (56 Horas) / ELE069945089677
    **Descripción del Módulo:** Este módulo proporciona los fundamentos esenciales para operar bases de datos relacionales, permitiendo a los participantes comprender y aplicar conceptos clave relacionados con la gestión, obtención, manipulación y definición de datos mediante el lenguaje estructurado de consultas (SQL). Se abordan aspectos fundamentales como la estructura, características y rol de las bases de datos relacionales dentro de las organizaciones. Los participantes aprenderán a realizar consultas efectivas utilizando SQL, aplicando sentencias condicionales, consultas multi-tabla, funciones de agrupación y tipos diversos de joins. También se exploran los lenguajes DML y DDL para modificar datos existentes, mantener integridad referencial, gestionar transacciones, y definir estructuras de datos robustas mediante creación, modificación y eliminación de tablas. Finalmente, el módulo cubre técnicas básicas de modelamiento de datos con diagramas entidad-relación, reglas de normalización y elaboración de diccionarios de datos para asegurar una gestión eficiente y efectiva de la información organizacional.
    
    **Competencia del Módulo:** Operar una base de datos relacional utilizando el lenguaje SQL para la obtención, manipulación y definición de datos dando solución a un problema de almacenamiento de información.
    
    Actividades: 46
    
- ### Módulo 6: Desarrollo de aplicaciones web con Python y Django (72 Horas) / ELE069945089678
    
    **Descripción del Módulo:** Este módulo aborda el desarrollo de aplicaciones web empresariales utilizando el framework Django en el entorno Python, enfatizando el patrón Modelo-Vista-Controlador (MVC). Los participantes aprenderán las características fundamentales de Django, cómo instalarlo y configurarlo dentro de un entorno virtual para mejorar la gestión y aislamiento de proyectos. Se profundiza en la creación y configuración inicial de proyectos y aplicaciones Django, incluyendo el manejo del servidor y rutas mediante archivos como manage.py y settings.py. Se enseña a desplegar contenido dinámico con templates, aplicando técnicas como herencia de plantillas y renderización condicional, así como la implementación efectiva de formularios reutilizables. Además, se exploran mecanismos avanzados de autenticación y autorización provistos por Django para asegurar aplicaciones, junto con el manejo eficiente del módulo administrativo para gestionar usuarios, grupos y permisos, fortaleciendo así la seguridad integral del desarrollo web empresarial.
    
    **Competencia del Módulo:** Construir aplicaciones web empresariales utilizando el patrón MVC en el entorno de desarrollo Python/Django para dar solución a los requerimientos de la organización.
    
    Actividades: 44
    
- ### Módulo 7: Acceso a datos en aplicaciones Python y Django (80 Horas) / ELE069945089679]
    **Descripción del Módulo:** Este módulo se centra en el acceso y manipulación de datos en aplicaciones web desarrolladas con Python y Django. Los participantes aprenderán a integrar bases de datos SQL mediante el uso del ORM (Object-Relational Mapping) que ofrece Django, permitiendo así realizar operaciones de creación, lectura, actualización y eliminación de datos (CRUD). El módulo aborda la definición y manejo de modelos de datos con y sin relaciones, incluyendo relaciones uno a uno, uno a muchos y muchos a muchos, facilitando una representación eficiente de estructuras complejas de información. También se trabajará en la gestión efectiva de cambios mediante migraciones y la realización de consultas avanzadas y personalizadas tanto con ORM como con sentencias SQL directas. Finalmente, se desarrollará una aplicación web MVC completa, integrando todos los conceptos anteriores, además de explorar aplicaciones preinstaladas en Django que optimizan y simplifican el proceso de desarrollo.
    
    **Competencia del Módulo:** Construir aplicaciones web que manipulan datos en una base de datos SQL utilizando Python/Django y las componentes que el lenguaje dispone para su uso para dar solución a un requerimiento.
    
    Actividades: 43
    
- ### Módulo 8: Desarrollo de portafolio de un producto digital (18 Horas) / ELE069945089680
    
    **Descripción del Módulo:** Este módulo tiene como objetivo proporcionar las habilidades necesarias para elaborar y presentar un portafolio efectivo que destaque productos digitales desarrollados por los participantes, potenciando así su perfil laboral. Se enfoca en la importancia estratégica del portafolio como herramienta profesional, abordando buenas prácticas disciplinarias y diferentes plataformas tecnológicas disponibles para su implementación, tales como GitHub, Behance, servicios de hosting y YouTube. Los participantes realizarán la revisión, depuración y mejora final de un producto digital desarrollado durante el curso, aplicando técnicas avanzadas y prácticas recomendadas para garantizar la calidad del entregable. Finalmente, aprenderán cómo presentar sus productos de forma atractiva y profesional en diversos formatos digitales para demostrar eficazmente sus capacidades técnicas ante posibles empleadores o clientes.
    
    **Competencia del Módulo:** Presentar un producto digital en un portafolio personal utilizando las herramientas tecnológicas y buenas prácticas disciplinares para resolver una problemática y potenciar el perfil laboral.
    
    Actividades: 22
    
- ### Módulo 9: Desarrollo de empleabilidad en la industria digital (18 Horas) / ELE069945089681
    
    **Descripción del Módulo:** Este módulo tiene como propósito principal preparar a los participantes para mejorar su empleabilidad en la industria digital, específicamente en el sector tecnológico (TI). Se aborda la elaboración efectiva de un plan de búsqueda laboral, considerando características específicas del mercado TI, tales como estructuras organizacionales, perfiles demandados, tipos de contratos y desafíos técnicos comunes. El módulo también proporciona herramientas para confeccionar y publicar un currículum vitae atractivo y relevante, además de técnicas para utilizar eficazmente plataformas laborales digitales. Asimismo, los participantes aprenderán a gestionar emociones, aplicar técnicas de escucha activa, preparar entrevistas laborales individuales y crear un elevator pitch convincente para destacar durante los procesos de selección en la industria TI.
    
    **Competencia del Módulo:** Elaborar un plan de búsqueda desarrollando el propio perfil profesional, reconociendo las características del mercado laboral del sector tecnológico y aplicando técnicas para la preparación de entrevistas.
    
    Actividades: 54
